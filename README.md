# Weighted-Feature-Distribution-Aligned-Data-Sampling-Approach
We propose a weighted feature-distribution aligned data sampling. The workflow can be summarized as “accurately model the data distribu- tion first, then adaptively learn features’ importance, after that intelligently optimize sample selection, last evaluate it”.
Weight-DQN.py is used for weighting featrues' impotance
approach_comparion.py reproduced other common data sampling methods and performed hyperparameter optimization to determine the optimal parameters for the experiments.
RL_select_samples.py is used for RL-based data sampling
